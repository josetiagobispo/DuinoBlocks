DUINOBLOCKS

DuinoBlocks is a visual programming language for Arduino using blocks to represent commands of the Wiring computer language (native language for Arduino).

The development of the software was partially granted by CNPq / Brazil . Project No. 550400/2011-7.

The software was developed as part of the Master dissertation of the first author supervised by the second author at Federal University of Rio de Janeiro (www.ppgi.ufrj.br)

Home: www.nce.ufrj.br/ginape/duinoblocks or www.duinoblocks.com.br


INSTALATION

Eclipse IDE (www.eclipse.org); Python Compiler 2.7 (www.python.org/downloads); PyDev Plugin (pydev.org); Pyjamas Library (pyjs.org)

For more details see https://github.com/pyjs/pyjs/wiki/usingeclipsewithpyjamas to setup those softwares.


USAGE

Copy only those files and folders: fonts, images, lang, output, scripts, sounds, stylesheets e index.html

Double click in file index.html to open DuinoBlocks on your browser (preferably in Chrome).
